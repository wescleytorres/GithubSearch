## Getting started

### - Pré-requisitos

- Node.js
- Yarn ou NPM
<br>
<br>

### - Instalando Dependencias
No diretório web inserir o comando: <br>
para npm
``` bash
npm install
```
para yarn
``` bash
yarn
```
<br>

### - Rodando aplicação
No diretório web inserir o comando: <br>
para npm
``` bash
npm run dev
```
para yarn
``` bash
yarn dev
```
